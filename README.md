# mcserver

Static join-guide for the Minecraft server.

## Server

- Address: `vanaj.pineserver.xyz`
- Port: `25525`
- Minecraft 1.21.1 / NeoForge 21.1.230

## Preview

```sh
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## Deploy

Push to the default branch. Netlify auto-deploys the repo root &mdash; no build step.
