The vanilla Minecraft launcher won't run modded servers. Follow the steps below to install Prism Launcher, drop in the mod pack, and connect.

  1. 01

    Install Java 21

    Required to run Minecraft 1.21.1. Skip if you already have it (java -version shows 21.x.x).

    • Temurin 21 (recommended) — pick your platform, run the installer.
    • brew install temurin@21 if you have Homebrew on macOS.
  2. 02

    Install Prism Launcher

    An open-source Minecraft launcher built for modded play.

    • Download Prism Launcher
    • On macOS: brew install --cask prismlauncher also works.
  3. 03

    First-time Prism setup

    • Open Prism, accept defaults. It'll auto-detect Java 21.
    • Top right → Accounts → Add Microsoft, sign in with your Minecraft account.
    • Top right → Settings (gear icon) → Java → set Maximum memory to 6144 MB (6 GB). Lower it if your machine has <16 GB RAM.
  4. 04

    Create the instance

    • Add Instance (top left) → Custom.
    • Name it anything — e.g. Modded SMP.
    • Minecraft version: 1.21.1
    • Mod Loader tab → NeoForge → version 21.1.230 → OK.

    Prism downloads and installs NeoForge automatically — no separate step needed.

  5. 05

    Download the mods

    Grab the mod pack from Drive. Everything you need is in the mods/ folder.

    Open mods on Drive →

    Download every .jar in mods/. Optionally grab config/ and defaultconfigs/ too — they keep mod settings consistent with the server.

  6. 06

    Drop the mods into Prism

    • Right-click your instance → Folder → Mods Folder.
    • Drag every .jar you downloaded into the folder that opens.
    • If you grabbed the config folders too: right-click instance → Folder → Instance Folder → open minecraft/ → drop in config/ and defaultconfigs/.
  7. 07

    Connect

    • Hit Launch. First boot takes 1–3 minutes.
    • From the title screen: Multiplayer → Add Server.
    • Server Address: vanaj.pineserver.xyz:25525
    • Save, double-click the server, you're in.

What's in the pack

The ones most likely to affect how you play:

  • Aether and Twilight Forest add two new dimensions, each with their own bosses and gear.
  • Origins lets you pick a species with unique abilities and drawbacks.
  • Point Blank adds guns and ammo.
  • Mystical Agriculture lets you grow resources from seeds.
  • Distant Horizons pushes view distance way past vanilla via low-detail LODs.
  • Waystones + JourneyMap handle fast travel and maps.
  • Dungeons Arise + Structory add large generated structures throughout the world.

New dimensions

  • Aether — A sky dimension above the overworld. Floating islands, blue clouds, new mobs (moas, valkyries, swets), and tiered dungeons (bronze, silver, gold) with bosses. Portal is built like a Nether portal but with glowstone, then filled with water to activate.
  • Deep Aether — Expansion for the Aether. Adds sub-dimensions inside the Aether, new biomes, and additional bosses. Requires the base Aether.
  • Twilight Forest — A dimension permanently locked at twilight. Has its own progression chain gated by defeating bosses: lich, naga, hydra, minotaur, snow queen, final dragon. Portal: dig a 2×2 hole, fill with water, ring it with flowers, then throw a diamond into the water.
  • Deeper Darker — Adds a dimension called the Otherside, themed around sculk. New blocks (echo wood, echo metal), new mobs (Stalkers, Sculk Snappers), new tools. Entry portal is hidden inside Ancient Cities.

Worldgen & structures

  • Oh The Biomes We've Gone — Adds 60+ overworld biomes alongside the vanilla ones (redwood forests, lush valleys, varied mountain types, deserts). Most overworld generation outside vanilla biomes comes from this mod.
  • Oh The Trees You'll Grow — Adds new tree types that populate the BWG biomes. Most non-vanilla trees you see are from this.
  • Dungeons Arise (+ Seven Seas) — Adds around 30 large hand-built structures spread across the overworld (sky pirate ships, monasteries, mushroom houses, etc.) and the ocean (shipwrecks, sea forts). Each is unique and contains mobs and loot.
  • Structory (+ Towers) — Adds smaller structures spread across biomes: ruins, shrines, abandoned bridges, and tall landmarks like lighthouses and watchtowers. Lower loot than Dungeons Arise; more for atmosphere.
  • Neo Medieval Buildings (+ End Edition) — Adds medieval-themed buildings (castles, taverns, ruins) that generate in the overworld, and a separate set that generates in the End dimension.

Mobs

  • Drakvyrn — Adds dragon-like creatures that spawn wild in specific biomes. You can tame them with meat, breed them, and ride them. Saddled adults can fly.
  • Friends and Foes — Adds the mobs that lost Mojang's mob votes (copper golem, illusioner, glare, moobloom, wildfire), plus variant villagers and small mob behavior changes.
  • Flowery Mooblooms — Adds variants of the moobloom mob, one per flower type. Cosmetic biodiversity.

Combat & sailing

  • Point Blank — Adds modern firearms: pistols, rifles, shotguns, sniper rifles with scopes. Guns craft from iron and redstone; ammo crafts from copper ingots and gunpowder. All recipes are viewable in JEI.
  • Smallships — Adds four buildable ship types (cog, brigantine, galley, drakkar). They sail with physics, carry cargo, and can mount cannons. You build them at a shipyard block, then place in water.

Farming & resources

  • Farmer's Delight — Adds cooking and farming mechanics on top of vanilla. New tools (knife, cutting board, cooking pot, skillet), new crops (tomato, cabbage, rice, onion, more), and meals that give status buffs when eaten.
  • Mystical Agriculture (+ Agradditions) — Adds a system for growing resources from seeds. Process: mine Prosperity Ore (spawns underground) to get shards, craft tier-1 Inferium Seeds, plant on farmland, harvest essence. Combine essence into the actual resource (iron, gold, diamond, etc.), or into higher-tier seeds. Tiers go Inferium → Prudentium → Tertium → Imperium → Supremium. Agradditions adds end-game tiers on top.
  • Botany Pots (+ Tiers) — Plant any crop, sapling, or flower in a pot. Grows without farmland, sunlight, or water. Tiers addon adds upgraded pots that grow faster and hold larger plants like trees.
  • Pylons — Adds decorative pylon blocks that give an area effect when activated (healing, harvest, water-walking, etc.). Crafted from gems and a core block.

Character

  • Origins — On first spawn (and on death), you pick an origin: a species with a unique ability set and drawbacks. E.g. Elytrian gets built-in elytra flight but takes hunger damage from solid food; Merling thrives in water but dries out on land. Open the selection screen with the keybind shown under Controls.
  • Accessories / Curios — Adds equipment slots beyond armor: rings, necklaces, belts, charms. Other mods drop items into these slots. The slot tab appears next to your inventory.

Storage & decor

  • Sophisticated Backpacks — Wearable backpacks. Each backpack can take upgrade modules: auto-pickup, magnet, filter, voiding, crafting, deposit/withdraw. Worn in the chest slot or in a curio slot.
  • Sophisticated Storage — Tiered storage barrels and chests using the same upgrade system as the backpacks. Wood tier first, upgrade through iron, gold, diamond, netherite for more slots.
  • Handcrafted — Furniture set: chairs, tables, lamps, cabinets, drawers, cushions, beds. Decorative only.
  • Chipped — Adds hundreds of decorative variants for vanilla blocks (wood, stone, glass, wool, etc.). Variants are crafted at new workbenches: carpenter's table, glassblower, loom table, mason's table.
  • Beautify — Vanilla-styled decoration set: shelves, hanging plants, lanterns, frames, food displays. Made to blend with vanilla textures.
  • Comforts — Adds sleeping bags and hammocks. Sleeping bags let you skip the night without resetting your spawn. Hammocks skip the day.

Travel & UI

  • Waystones — Place a waystone block, name it. Right-click any other waystone to teleport to it (no item cost by default). The most reliable fast-travel mod.
  • JourneyMap — Adds a minimap (corner of screen), a full-screen world map (default key: J), waypoint markers, and a browser-based map view. Auto-marks waystones you've used.
  • JEI (Just Enough Items) — Adds a sidebar in the inventory screen listing every item in the game. Type to search. Click an item to see its recipe; click the output to see where it's used.
  • WTHIT — Shows a small tooltip at the top of the screen describing whichever block or mob your crosshair is on, including which mod it's from.

Visual

  • Distant Horizons — Renders far-away terrain at low detail (LODs) so you can see way past vanilla's view distance without tanking frame rate. LODs are cached locally as you explore. Client-side; the server doesn't know it exists.

Quality of life

  • FallingTree — Hit the bottom log of a tree with an axe and the whole tree breaks at once instead of one log at a time.
  • Fast Leaf Decay — After a tree is cut, the leaves decay in 2–3 seconds instead of vanilla's ~5 minutes.
  • FTB Ultimine — Hold the keybind (default: `) and break a block to mine the entire connected vein at once. Works on ores, logs, leaves, stone — any block with connected neighbors of the same type.
  • Double Doors — Opens both halves of a double door, trapdoor, or fence gate together. Works for clicks, pressure plates, and buttons.

Troubleshooting

"Not enough RAM available"

Prism is asking for more memory than your machine has free. Quit memory-hungry apps, then check Activity Monitor / Task Manager for free memory. If you only have 8 GB total, lower the Max in Prism → Java to 3072 MB.

"Failed to download mod" / Drive download blocked

Google Drive rate-limits large folder downloads. Download files in smaller batches, or wait a few hours and try again. As a last resort, ask me for a direct mirror link.

"Failed to connect" / "Connection refused"

Server may be offline or restarting. Wait a minute and retry. If it keeps failing, let me know.

Crash on launch / where are the logs?

Open Prism, right-click the instance → Logs. Or find them at:

  • macOS: ~/Library/Application Support/PrismLauncher/instances/<name>/minecraft/logs/latest.log
  • Windows: %APPDATA%\PrismLauncher\instances\<name>\minecraft\logs\latest.log
