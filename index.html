The vanilla Minecraft launcher won't run modded servers. Follow the steps below to install Prism Launcher, drop in the mod pack, and connect.

  1. 01

    Install Java 21

    Required to run Minecraft 1.21.1. Skip if you already have it (java -version shows 21.x.x).

    • Temurin 21 (recommended) — pick your platform, run the installer.
    • brew install temurin@21 if you have Homebrew on macOS.
  2. 02

    Install Prism Launcher

    An open-source Minecraft launcher built for modded play.

    • Download Prism Launcher
    • On macOS: brew install --cask prismlauncher also works.
  3. 03

    First-time Prism setup

    • Open Prism, accept defaults. It'll auto-detect Java 21.
    • Top right → Accounts → Add Microsoft, sign in with your Minecraft account.
    • Top right → Settings (gear icon) → Java → set Maximum memory to 6144 MB (6 GB). Lower it if your machine has <16 GB RAM.
  4. 04

    Create the instance

    • Add Instance (top left) → Custom.
    • Name it anything — e.g. Modded SMP.
    • Minecraft version: 1.21.1
    • Mod Loader tab → NeoForge → version 21.1.95 → OK.

    Prism downloads and installs NeoForge automatically — no separate step needed.

  5. 05

    Download the mods

    Grab the mod pack from Drive. Everything you need is in the mods/ folder.

    Open mods on Drive →

    Download every .jar in mods/. Optionally grab config/ and defaultconfigs/ too — they keep mod settings consistent with the server.

  6. 06

    Drop the mods into Prism

    • Right-click your instance → Folder → Mods Folder.
    • Drag every .jar you downloaded into the folder that opens.
    • If you grabbed the config folders too: right-click instance → Folder → Instance Folder → open minecraft/ → drop in config/ and defaultconfigs/.
  7. 07

    Connect

    • Hit Launch. First boot takes 1–3 minutes.
    • From the title screen: Multiplayer → Add Server.
    • Server Address: vanaj.pineserver.xyz:25525
    • Save, double-click the server, you're in.

Troubleshooting

"Not enough RAM available"

Prism is asking for more memory than your machine has free. Quit memory-hungry apps, then check Activity Monitor / Task Manager for free memory. If you only have 8 GB total, lower the Max in Prism → Java to 3072 MB.

"Failed to download mod" / Drive download blocked

Google Drive rate-limits large folder downloads. Download files in smaller batches, or wait a few hours and try again. As a last resort, ask me for a direct mirror link.

"Failed to connect" / "Connection refused"

Server may be offline or restarting. Wait a minute and retry. If it keeps failing, ping in the group chat.

Crash on launch / where are the logs?

Open Prism, right-click the instance → Logs. Or find them at:

  • macOS: ~/Library/Application Support/PrismLauncher/instances/<name>/minecraft/logs/latest.log
  • Windows: %APPDATA%\PrismLauncher\instances\<name>\minecraft\logs\latest.log
